Image Generator API Guide

1. Single Image Generation

Endpoint: /{width}/{height}

Generate a single image with specified dimensions.

Parameters:

width: Image width (1-3000 pixels)

height: Image height (1-3000 pixels)

text: Optional text to display (supports Unicode)

color: Background color in hex format (e.g., FF0000 or #FF0000)

Examples:

Basic image: /300/200

With text: /300/200?text=Hello

With color: /300/200?color=FF0000

With text and color: /300/200?text=Hello&color=FF0000

2. Bulk Image Generation

Endpoint: /bulk/{count}/{width}/{height}

Generate multiple images at once.

Parameters:

count: Number of images to generate (1-50)

width: Image width (1-3000 pixels)

height: Image height (1-3000 pixels)

text: Optional text to display

color: Background color in hex format

samebg: Use same background color for all images (true/false)

numbering: Add numbers to text (true/false)

download: Download as ZIP file (true/false)

Examples:

Basic bulk: /bulk/5/300/200

With text: /bulk/5/300/200?text=Hello

With numbering: /bulk/5/300/200?text=Hello&numbering=true

Same background: /bulk/5/300/200?text=Hello&samebg=true

Download ZIP: /bulk/5/300/200?text=Hello&download=true
