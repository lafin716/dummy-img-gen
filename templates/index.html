🎨 Dummy Image Generator

Welcome to the Dummy Image Generator! This tool allows you to generate placeholder images with custom dimensions, text, and colors.

Basic Usage

Generate an image by specifying width and height:

http://localhost:8000/300/200

Optional Parameters

Custom Text

Add custom text above the dimensions using the text parameter:

http://localhost:8000/300/200?text=Hello%20World

Background Color

Set a specific background color using the color parameter with a hex color code:

http://localhost:8000/300/200?color=FF5733
Note: If the color parameter is invalid or not provided, a random color will be used.

Combining Parameters

You can combine multiple parameters:

http://localhost:8000/300/200?text=Hello%20World&color=FF5733

Features

  • Custom dimensions (up to 3000x3000 pixels)
  • Optional custom text display
  • Hex color background support
  • Random color generation
  • Automatic text sizing and positioning
  • High contrast text with outline for readability

🎨 ダミー画像ジェネレーター

ダミー画像ジェネレーターへようこそ！このツールでカスタムサイズ、テキスト、色の画像を生成できます。

基本的な使い方

幅と高さを指定して画像を生成:

http://localhost:8000/300/200

オプションパラメータ

カスタムテキスト

textパラメータを使用してサイズの上にテキストを追加:

http://localhost:8000/300/200?text=こんにちは

背景色

hexコードでcolorパラメータを使用して特定の背景色を設定:

http://localhost:8000/300/200?color=FF5733

パラメータの組み合わせ

複数のパラメータを組み合わせることができます:

http://localhost:8000/300/200?text=こんにちは&color=FF5733

機能

  • カスタムサイズ（最大3000x3000ピクセル）
  • オプションのカスタムテキスト表示
  • HEXカラーコードによる背景色設定
  • ランダムカラー生成
  • 自動テキストサイズ調整と配置
  • 読みやすさを考慮した輪郭付きの高コントラストテキスト

🎨 示例图片生成器

欢迎使用示例图片生成器！此工具可以生成自定义尺寸、文本和颜色的图片。

基本用法

通过指定宽度和高度生成图片：

http://localhost:8000/300/200

可选参数

自定义文本

使用text参数在尺寸上方添加文本：

http://localhost:8000/300/200?text=你好

背景颜色

使用color参数设置特定的背景颜色（十六进制代码）：

http://localhost:8000/300/200?color=FF5733
注意：如果颜色参数无效或未提供，将使用随机颜色。

组合参数

您可以组合多个参数：

http://localhost:8000/300/200?text=你好&color=FF5733

功能特点

  • 自定义尺寸（最大3000x3000像素）
  • 可选的自定义文本显示
  • 支持十六进制颜色代码
  • 随机颜色生成
  • 自动文本大小调整和定位
  • 高对比度文本，带有轮廓以提高可读性

🎨 Trình Tạo Ảnh Mẫu

Chào mừng đến với Trình Tạo Ảnh Mẫu! Công cụ này cho phép bạn tạo ảnh với kích thước, văn bản và màu sắc tùy chỉnh.

Cách Sử Dụng Cơ Bản

Tạo ảnh bằng cách chỉ định chiều rộng và chiều cao:

http://localhost:8000/300/200

Tham Số Tùy Chọn

Văn Bản Tùy Chỉnh

Thêm văn bản phía trên kích thước bằng tham số text:

http://localhost:8000/300/200?text=Xin%20chào

Màu Nền

Đặt màu nền cụ thể bằng tham số color với mã màu hex:

http://localhost:8000/300/200?color=FF5733
Lưu ý: Nếu tham số màu không hợp lệ hoặc không được cung cấp, một màu ngẫu nhiên sẽ được sử dụng.

Kết Hợp Tham Số

Bạn có thể kết hợp nhiều tham số:

http://localhost:8000/300/200?text=Xin%20chào&color=FF5733

Tính Năng

  • Kích thước tùy chỉnh (tối đa 3000x3000 pixel)
  • Hiển thị văn bản tùy chỉnh tùy chọn
  • Hỗ trợ mã màu hex cho nền
  • Tạo màu ngẫu nhiên
  • Tự động điều chỉnh kích thước và vị trí văn bản
  • Văn bản tương phản cao với viền để dễ đọc
