Welcome to the Dummy Image Generator! This tool allows you to generate placeholder images with custom dimensions, text, and colors.

Basic Usage

Generate an image by specifying width and height:

http://localhost:8000/300/200

Optional Parameters

Custom Text

Add custom text above the dimensions using the text parameter:

http://localhost:8000/300/200?text=Hello%20World

Background Color

Set a specific background color using the color parameter with a hex color code:

http://localhost:8000/300/200?color=FF5733
Note: If the color parameter is invalid or not provided, a random color will be used.

Combining Parameters

You can combine multiple parameters:

http://localhost:8000/300/200?text=Hello%20World&color=FF5733

Features

  • Custom dimensions (up to 3000x3000 pixels)
  • Optional custom text display
  • Hex color background support
  • Random color generation
  • Automatic text sizing and positioning
  • High contrast text with outline for readability
