Image Generator API Guide

1. Single Image Generation

Endpoint: /{width}/{height}

Generate a single image with specified dimensions.

Parameters:

width: Image width (1-3000 pixels)

height: Image height (1-3000 pixels)

text: Optional text to display (supports Unicode)

color: Background color in hex format (e.g., FF0000 or #FF0000)

Examples:

Basic image: /300/200

With text: /300/200?text=Hello

With color: /300/200?color=FF0000

With text and color: /300/200?text=Hello&color=FF0000

2. Bulk Image Generation

Endpoint: /bulk/{count}/{width}/{height}

Generate multiple images at once.

Parameters:

count: Number of images to generate (1-50)

width: Image width (1-3000 pixels)

height: Image height (1-3000 pixels)

text: Optional text to display

color: Background color in hex format

samebg: Use same background color for all images (true/false)

numbering: Add numbers to text (true/false)

download: Download as ZIP file (true/false)

Examples:

Basic bulk: /bulk/5/300/200

With text: /bulk/5/300/200?text=Hello

With numbering: /bulk/5/300/200?text=Hello&numbering=true

Same background: /bulk/5/300/200?text=Hello&samebg=true

Download ZIP: /bulk/5/300/200?text=Hello&download=true

1. 단일 이미지 생성

엔드포인트: /{width}/{height}

지정된 크기의 단일 이미지를 생성합니다.

파라미터:

width: 이미지 너비 (1-3000 픽셀)

height: 이미지 높이 (1-3000 픽셀)

text: 표시할 텍스트 (선택사항, 유니코드 지원)

color: 배경색 (16진수 형식, 예: FF0000 또는 #FF0000)

예시:

기본 이미지: /300/200

텍스트 포함: /300/200?text=안녕하세요

배경색 지정: /300/200?color=FF0000

텍스트와 배경색: /300/200?text=안녕하세요&color=FF0000

2. 대량 이미지 생성

엔드포인트: /bulk/{count}/{width}/{height}

여러 개의 이미지를 한 번에 생성합니다.

파라미터:

count: 생성할 이미지 수 (1-50)

width: 이미지 너비 (1-3000 픽셀)

height: 이미지 높이 (1-3000 픽셀)

text: 표시할 텍스트 (선택사항)

color: 배경색 (16진수 형식)

samebg: 모든 이미지에 동일한 배경색 사용 (true/false)

numbering: 텍스트에 번호 추가 (true/false)

download: ZIP 파일로 다운로드 (true/false)

예시:

기본 대량 생성: /bulk/5/300/200

텍스트 포함: /bulk/5/300/200?text=안녕하세요

번호 추가: /bulk/5/300/200?text=안녕하세요&numbering=true

동일 배경색: /bulk/5/300/200?text=안녕하세요&samebg=true

ZIP 다운로드: /bulk/5/300/200?text=안녕하세요&download=true

1. 単一画像生成

エンドポイント: /{width}/{height}

指定されたサイズの単一画像を生成します。

パラメータ:

width: 画像の幅 (1-3000 ピクセル)

height: 画像の高さ (1-3000 ピクセル)

text: 表示するテキスト (オプション、Unicode対応)

color: 背景色 (16進数形式、例: FF0000 または #FF0000)

例:

基本画像: /300/200

テキスト付き: /300/200?text=こんにちは

背景色指定: /300/200?color=FF0000

テキストと背景色: /300/200?text=こんにちは&color=FF0000

2. 一括画像生成

エンドポイント: /bulk/{count}/{width}/{height}

複数の画像を一度に生成します。

パラメータ:

count: 生成する画像数 (1-50)

width: 画像の幅 (1-3000 ピクセル)

height: 画像の高さ (1-3000 ピクセル)

text: 表示するテキスト (オプション)

color: 背景色 (16進数形式)

samebg: すべての画像に同じ背景色を使用 (true/false)

numbering: テキストに番号を追加 (true/false)

download: ZIPファイルとしてダウンロード (true/false)

例:

基本一括生成: /bulk/5/300/200

テキスト付き: /bulk/5/300/200?text=こんにちは

番号付き: /bulk/5/300/200?text=こんにちは&numbering=true

同一背景色: /bulk/5/300/200?text=こんにちは&samebg=true

ZIPダウンロード: /bulk/5/300/200?text=こんにちは&download=true

1. 单个图像生成

端点: /{width}/{height}

生成指定尺寸的单个图像。

参数:

width: 图像宽度 (1-3000 像素)

height: 图像高度 (1-3000 像素)

text: 显示文本 (可选，支持Unicode)

color: 背景颜色 (十六进制格式，例如: FF0000 或 #FF0000)

示例:

基本图像: /300/200

带文本: /300/200?text=你好

指定背景色: /300/200?color=FF0000

文本和背景色: /300/200?text=你好&color=FF0000

2. 批量图像生成

端点: /bulk/{count}/{width}/{height}

一次生成多个图像。

参数:

count: 生成图像数量 (1-50)

width: 图像宽度 (1-3000 像素)

height: 图像高度 (1-3000 像素)

text: 显示文本 (可选)

color: 背景颜色 (十六进制格式)

samebg: 所有图像使用相同背景色 (true/false)

numbering: 文本添加编号 (true/false)

download: 下载为ZIP文件 (true/false)

示例:

基本批量: /bulk/5/300/200

带文本: /bulk/5/300/200?text=你好

带编号: /bulk/5/300/200?text=你好&numbering=true

相同背景: /bulk/5/300/200?text=你好&samebg=true

ZIP下载: /bulk/5/300/200?text=你好&download=true

1. Tạo Ảnh Đơn Lẻ

Endpoint: /{width}/{height}

Tạo một ảnh với kích thước được chỉ định.

Tham số:

width: Chiều rộng ảnh (1-3000 pixel)

height: Chiều cao ảnh (1-3000 pixel)

text: Văn bản hiển thị (tùy chọn, hỗ trợ Unicode)

color: Màu nền (định dạng hex, ví dụ: FF0000 hoặc #FF0000)

Ví dụ:

Ảnh cơ bản: /300/200

Với văn bản: /300/200?text=Xin chào

Với màu nền: /300/200?color=FF0000

Văn bản và màu: /300/200?text=Xin chào&color=FF0000

2. Tạo Nhiều Ảnh

Endpoint: /bulk/{count}/{width}/{height}

Tạo nhiều ảnh cùng một lúc.

Tham số:

count: Số lượng ảnh cần tạo (1-50)

width: Chiều rộng ảnh (1-3000 pixel)

height: Chiều cao ảnh (1-3000 pixel)

text: Văn bản hiển thị (tùy chọn)

color: Màu nền (định dạng hex)

samebg: Sử dụng cùng màu nền cho tất cả ảnh (true/false)

numbering: Thêm số thứ tự vào văn bản (true/false)

download: Tải xuống dưới dạng file ZIP (true/false)

Ví dụ:

Cơ bản: /bulk/5/300/200

Với văn bản: /bulk/5/300/200?text=Xin chào

Với số thứ tự: /bulk/5/300/200?text=Xin chào&numbering=true

Cùng màu nền: /bulk/5/300/200?text=Xin chào&samebg=true

Tải ZIP: /bulk/5/300/200?text=Xin chào&download=true
